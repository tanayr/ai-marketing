# Indie Kit

Indie Kit is a NextJS Boilerplate for building SaaS products.

## Features

- NextJS 15

## Getting Started

1. Clone the repo
2. Run `pnpm install`
3. Run `pnpm dev`

For detailed documentation, visit [https://indiekit.pro/app/docs](https://indiekit.pro/app/docs)
